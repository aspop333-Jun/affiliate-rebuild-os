# Affiliate Rebuild OS Overview

このリポジトリは **「未インデックス資産を、Googleに通る形で再起動するための運用OS」** です。

## 目的
未インデックス状態の既存資産（約5000記事/サイト）を、スクラップ＆ビルド（非公開→新規公開）で再投入し、**Googleに「人が運営しているサイト」として自然に受理される投稿運用の型**を確立します。

## 主要ディレクトリ
- `/docs`: ルール・手順書の正本
- `/prompts`: AIへの指示書（プロンプト）
- `/sites`: サイトごとの記事データ・進捗管理
- `/scripts`: 自動化ツール

## 運用フロー概要
1. **Scrap**: 旧記事を非公開化
2. **Re-design**: 検索意図と読者に合わせて構成を再設計
3. **Draft (AI)**: AIが構造と下書きを作成
4. **Rewrite (Human)**: 人間が「キャンディ・コタロー調」にリライト
5. **Review**: 公開前チェック（AI臭の除去）
6. **Publish**: 新規URLとして公開
7. **Observe**: Search Consoleでインデックス状況を観測

詳細なルールは `docs/01_rules/` を参照してください。
