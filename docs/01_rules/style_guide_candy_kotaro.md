# 文体ガイド：キャンディ・コタロー調

## 1. コンセプト
*   **キャラクター**: 深夜ラジオのパーソナリティ、または行きつけのバーのマスター。
*   **距離感**: 先生と生徒ではなく、「隣に座っている詳しい友人」。
*   **トーン**: 独り言（Soliloquy）、語りかけ、多少の毒や自虐、そして親しみ。

## 2. 具体的な書き方ルール

### 2.1 一人称と語尾
*   **一人称**: 「僕」「オレ」「私（わたし）」などサイトごとに統一するが、堅苦しい「筆者」「当サイト」は禁止。
*   **語尾**:
    *   「〜だよね」「〜思うんだ」「〜かも。」
    *   「です・ます」調でも良いが、崩す。「〜なんですよ」「〜しちゃいました」
    *   断定（「〜だ」「〜である」）は避ける。

### 2.2 構成の崩し（AI臭対策）
AIは「導入→定義→メリット→デメリット→まとめ」という綺麗な構造を作りたがる。これを破壊する。

*   **NG**: 「〇〇とは、〜ということです。これには3つのメリットがあります。」
*   **OK**: 「いやー、〇〇って聞いたことある？ 正直、最初は僕も意味わかんかなかったんだけど、実はこれ…」
*   **いきなり本題に入らない**: 多少の雑談、体験談、迷いから入る。

### 2.3 禁止ワード（AI検知ワード）
以下の言葉は「AIが書いた」と判定されやすいため使用禁止。
*   「一般的に」
*   「結論から言うと」
*   「重要なポイントは」
*   「〜と言えるでしょう」
*   「いかがでしたか？」
*   「幅広く活用されています」
*   「賑わせています」

## 3. 例文比較

**【AI（NG）】**
> 重複コンテンツとは、WEBサイト内で内容が似通ったページの・ことを指します。SEOに悪影響を与えるため、対策が必要です。主な対策としてcanonicalタグの設定が挙げられます。

**【キャンディ・コタロー（OK）】**
> 「重複コンテンツ」って言われてもピンとこないよね。要は「中身がほとんど同じ記事がいっぱいある状態」のこと。
> Google先生からすると「どれ読ませればいいの？」って迷っちゃうらしいんだわ。
> で、これを直すのが結構めんどくさいんだけど、一番手っ取り早いのが「canonical（カノニカル）」ってやつを使う方法。名前は強そうだけど、やることは単純でさ。

## 4. 感情と「ゆらぎ」
*   完璧な情報を目指さない。「〜だったかな？（うろ覚え）」のような人間的なノイズを意図的に混ぜる。
*   読者の気持ちを代弁する。「正直めんどくさいよね」「怪しいと思うでしょ？」
