# 投稿ポリシー (Publishing Policy)

## 1. 投稿ペース制限
Googleに「スパム的な量産」と判定されるのを防ぐため、以下のペースを厳守する。

*   **1サイトあたり：1日最大4記事**
*   **絶対禁止事項**:
    *   短時間での集中投稿（例：1時間に4本すべて公開）
    *   ツールによる機械的な自動投稿（cronで同時刻に投稿など）

## 2. 時間分散ルール
投稿時間は自然な「人間が活動している時間帯」に分散させる。

*   **推奨タイムスロット**:
    *   朝: 08:00 - 10:00
    *   昼: 12:00 - 14:00
    *   夕: 17:00 - 19:00
    *   夜: 21:00 - 23:00
*   **ゆらぎ**:
    *   毎回「09:00」ジャストにするのではなく、「09:12」「09:45」のように分単位でずらすこと。

## 3. 非公開・再公開ルール (Scrap & Build)
*   **原則**: 既存の未インデックス記事は、WordPress上で「下書き（Draft）」または「非公開（Private）」に戻す。
*   **URL変更**: 再公開時は **新しいURL（スラッグ）** を発行する。
    *   理由：Googleに「新しい記事」として認識させるため。
    *   旧URLは404にするか、新URLへ301リダイレクト（ケースバイケースだが、今回は「新規評価」を得るため404またはリダイレクトなしで新規投稿を推奨）。

## 4. インデックス確認
*   公開後、必ず Google Search Console (GSC) の「URL検査」を行い、インデックス登録リクエストを送る（初期フェーズ）。
*   48〜72時間後にGSCを確認し、結果を `publish_log.md` に記録する。
