# 公開前レビューチェックリスト

## 1. 必須チェック（Safety）
- [ ] **誤情報・ハルシネーションがないか**：AIがもっともらしい嘘をついていないか事実確認したか？
- [ ] **コンプライアンス**：Googleポリシー違反（過激なアダルト表現、暴力、誹謗中傷）が含まれていないか？
- [ ] **リンク切れ**：内部リンク・外部リンクは正しく機能しているか？

## 2. 脱AI臭チェック（Quality）
- [ ] **導入文の破壊**：「〇〇とは〜」という定義から始まっていないか？ 独自の語り出しになっているか？
- [ ] **接続詞の多様性**：「また」「さらに」「したがって」ばかり使っていないか？ 「ていうか」「ぶっちゃけ」などの崩しが入っているか？
- [ ] **綺麗すぎる箇条書き**：理由やメリットが毎回きれいな箇条書きになっていないか？（文章で語る部分を作る）
- [ ] **まとめの定型文**：「いかがでしたか」「〜を活用していきましょう」で終わっていないか？

## 3. 人間味チェック（Engagement）
- [ ] **一人称の語り**：運営者の「顔」が見える文章になっているか？
- [ ] **感情の吐露**：「驚いた」「苦労した」「笑った」などの感情表現があるか？
- [ ] **読者への問いかけ**：一方的な講義ではなく、会話になっているか？

## 4. 構造チェック（SEO）
- [ ] **タイトル**：検索意図を含みつつ、クリックしたくなる魅力があるか？
- [ ] **見出し**：h2/h3の構成は論理的か？（AI任せだとh2とh3の粒度が狂うことがある）
- [ ] **装飾**：太字、マーカー、ボックスが適切に使われ、読みやすくなっているか？
