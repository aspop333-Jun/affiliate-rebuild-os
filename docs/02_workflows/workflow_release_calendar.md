# Workflow: Release Calendar

## 1. カレンダー作成の仕組み
*   スクリプト `scripts/ops/generate_release_calendar.py` を使用して自動生成する。
*   **入力**: 開始日、記事数
*   **出力**: CSVファイル (`release_plan.csv`)

## 2. 運用ルール
*   **原則**: 生成されたカレンダーの日時に従って予約投稿を行う。
*   **調整**:
    *   急ぎの記事がある場合は、空いているスロットと入れ替える。
    *   **決して「1日4本」の上限を超えて詰め込まないこと**。
